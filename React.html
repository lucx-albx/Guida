Installazione

Partiamo con l'installazione
Per poter iniziare a programmare in React dobbiamo prima scaricare Node.js, per poterlo scaricare ti puoi recare sul loro  sito ufficiale, io ti consiglio di scaricare la LTS, ovvero l'ultima versione disponibile di Node.

Creazione di una Pagina

Come inizare
Una volta scaricato Node.js puoi aprire il tuo prompt di comandi e digitare:
npx create-react-app NomeCheVuoi
questo comando ti crereà la cartella con tutto il necessario al suo interno. Una volta aperta la cartella ci troviamo di fronte 3 cartelle, la prima sarà node-modules, come ti consiglia il nome ci saranno tutti i moduli necessari per ora, la seconda cartella sarà public, ovvero la cartella che contiene index.html della nostra pagina e infine src, la cartella che conterrà il nostro progetto, quindi cliccala e ti ritroverai di fronte a te molti file ma quelli che ci interessano sono principalmente App.js e App.css, quindi aprili. in App.js ti troverai di base una pagina d'esempio di questo genere:
Come funziona App.js
Volendo potreasti cancellare tutto, e se hai installato l'estensione consigliata all'inizio, digitare rafce, in poche parole ti ricrea tutto il necessario. Ma prima di continuare vediamo che in react a differenza dell'html per fare una classe si usa className e non class, l'altra differenza e che il nostro codice che una volta lo andavamo a scrivere nel body si scrive nel return, ma perchè nel return? il return usa il linguaggio JSX (javaScript Extension) e quindi al suo interno andrà la nosta "pagina", Mentre tra {} ci va javascipt, mentre sopra il return ci andranno le nostre funzione ecc.. Capito ciò per visualizzare il nostro progetto apriamo il prompt dei comandi e andiamo a digitare npm start, ed ecco visualizzato il nostro progetto!

Hooks

Cos'è un Hooks
Gli Hooks ti permettono di utilizzare state ed altre funzioni di React senza dover scrivere una classe. Gli Hooks che andremo a visionare in questa guida saranno:
- UseState
- UseEffect
Cos'è useState
UseState come ho detto prima è un Hooks che ci permette di andar aggiornare il nostro componente. Per usare questo Hooks dovremmo prima andarlo ad importare e poi digitare:
const [Per vedere cosa c'e all'interno dello stato, Serve per modificare lo stato (è UNA FUNZIONE)] = useState("") --> memorizzo solo UN dato!(Più dati = [])
nella quadre si andranno a mettere due parametri divisi da una virgola = useState("") se andiamo a mettere i doppi apici come scritto sopra memorizza un solo dato mentre con le [] andrà a memorizzare più dati.
Esempio di useState:
                        
    import React, { useState } from 'react'

    const App =()=>{
        const [MsgPera, setMsgPera] = useState([])
    
        const URL = "https://raw.githubusercontent.com/icobasco/sample_data_files/master/pera_misure_sample.json"
    
        const scarica =()=>{
            fetch(URL)
            .then((testo)=>testo.json())
            .then((pera)=>{
                setMsgPera(pera[0].data.sensor1.lowRes.humidity + " " + pera[0].data.sensor1.lowRes.temperature)
            
            })
        }
    
        return (
            <div>{MsgPera} Prova{scarica()}</div>
        )
    }
