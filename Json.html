Cos'è il JSON?

Definizione di JSON

JSON è un formato di serializzazione basato su testo per lo scambio di dati, principalmente tra server e applicazione web. JSON è l'acronimo di JavaScript Object Notation, e utilizza l'estensione di file . json. JSON è un concorrente di XML, ma possiede una sintassi più semplice e compatta rispetto ad esso.

#1 Sintassi Json
Una sintassi del json e ad oggetto che è una serie non ordinata di nomi o valori. Un oggetto inizia con {parentesi graffa sinistra e finisce con }parentesi graffa destra. Ogni nome è seguito da :due punti e la coppia di nome o valore sono separata da ,virgola

- La Fetch

Fetch

Utilizzo della Fetch

La FETCH tramite l'utilizzo di un URL è in grado di estrapolare i dati da un file .json. La fetch utilizza due .then() nel primo si va "selezionare" il JSON e nella seconda vengono presi i suoi dati, e all'intero di essa viene fatto un callBack per poi andare a visualizzare questi dati nel nostro sito. Ma vediamo un esempio!

Esempio Fetch:
                    
    const url = "https://raw.githubusercontent.com/icobasco/sample_data_files/master/pera_misure_sample.json"

    const datiLoading =()=>{
        fetch(url)
        .then((dati) => dati.json())
        .then((data)=>{
            const dateTable = document.querySelector("table tbody");
    
            let insert = ""
            data.forEach(function({lora_device_id, data, measured_at}) {
    
                insert += "<tr>";
                insert += `<td>${lora_device_id}</td>`
                insert += `<td>${measured_at}</td>`
                insert += `<td>${data.sensor1.lowRes.humidity}</td>`
                insert += `<td>${data.sensor1.lowRes.humidity}</td>`
                insert += "<tr>";
        
            })
            dateTable.innerHTML = insert;
        })
    }
